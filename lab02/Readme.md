Бочков Андрей

## Используемые технологии  
-**MongoDB** хранение каталога фильмов и метаданных контента  
-**GraphDB** работа с RDF-графом и SPARQL-запросами  
-**JupyterLab / Python** визуализация результатов и бизнес-аналитика  

## Выполненные задания
1. **MongoDB:** создана коллекция `movies_meta`, добавлены документы с вложенными массивами жанров, выполнен поиск фильмов с жанром `Drama`, выполнено обновление документа.
2. **GraphDB / SPARQL:** создан репозиторий `movies_repo`, загружен RDF-датасет `movies.ttl`, получены фильмы 90-х с участием Keanu Reeves, отсортированные по количеству комментариев.
3. **Аналитика:** построена диаграмма популярности фильмов Keanu Reeves 90-х годов на основе числа комментариев.
